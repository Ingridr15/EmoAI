<h1 style="color: #5a32e2;">EmoAI</h1>

<h2>Descripción</h2>
<p>EmoAI es una aplicación de detección de emociones en imágenes en tiempo real. Utiliza un modelo de inteligencia artificial entrenado para detectar emociones como enojo, felicidad, tristeza, etc., en rostros humanos capturados por una cámara.</p>

<h2>Instalación</h2>
<ol>
  <li>Clona este repositorio en tu máquina local:</li>
  <pre><code>git clone https://github.com/tu-usuario/EmoAI.git</code></pre>
  <li>Instala las dependencias ejecutando el siguiente comando:</li>
  <pre><code>pip install -r requirements.txt</code></pre>
</ol>

<h2>Uso</h2>
<p>Una vez que la aplicación esté en funcionamiento, apunta tu cámara a un rostro humano y la aplicación detectará automáticamente la emoción en tiempo real.</p>

<img src="screenshots/screenshot.png" alt="Ejemplo de la aplicación en funcionamiento" style="width: 50%; border: 2px solid #5a32e2; margin-bottom: 20px;">

<h2>Datos y Modelos</h2>
<p>El modelo pre-entrenado y los archivos de datos de imágenes están disponibles en la carpeta <code>Model</code> y <code>Pictures</code> respectivamente. Asegúrate de descargarlos y colocarlos en las ubicaciones especificadas.</p>
